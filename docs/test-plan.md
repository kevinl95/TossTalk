# Test Plan

## Motion scenarios

- Toss + catch
- Toss + bounce + pickup
- Desk bump
- Walking while speaking

## Pass criteria

- Device auto-recovers with no user interaction
- Talk availability remains high after toss cycles
- Throw-noise suppression is present but not prioritized over speech continuity

## Battery scenarios

- Discharging run: HUD remains visible and updates
- Charging plug/unplug: charging indicator updates
- Reboot: battery HUD appears immediately after startup
